## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or pnpm

### Installation

1. Clone the repository
```bash
git clone https://github.com/maitreyouss/qr-code-component-main.git
cd qr-code-component-main
```

2. Install dependencies
```bash
npm install
```

3. Start Tailwind CSS in watch mode (development)
```bash
npm run dev
```

4. Open `index.html` in your browser or use a live server

### Build for production

This will compile and minify the Tailwind CSS output:
```bash
npm run build
```
